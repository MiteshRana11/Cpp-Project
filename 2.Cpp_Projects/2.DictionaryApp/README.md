# Dictionary App (C++)

A console-based dictionary application that allows users to add, search, and delete words with their meanings. Data is stored using file handling so that the dictionary persists between program executions.

## Features

* Add new words
* Search word meanings
* Delete words
* Display dictionary
* Persistent storage using files

## Concepts Used

* Object-Oriented Programming
* STL (unordered_map)
* File Handling
* Encapsulation

## How to Run

Compile:

g++ DA.cpp -o dictionary

Run:

./dictionary

## Learning Outcome

This project demonstrates how STL containers and file handling can be combined to build a simple data management application.
