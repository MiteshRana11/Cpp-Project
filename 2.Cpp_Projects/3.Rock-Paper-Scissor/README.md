# Rock Paper Scissors Game (C++)

A console-based implementation of the classic Rock Paper Scissors game where the player competes against the computer.

## Features

* Player vs Computer gameplay
* Random computer choice
* Score tracking
* Continuous game loop

## Concepts Used

* Classes
* Enums
* Random number generation
* Game logic implementation

## How to Run

Compile:

g++ RPS.cpp -o game

Run:

./game

## Learning Outcome

This project demonstrates how to implement game logic and randomness using modern C++ features.
