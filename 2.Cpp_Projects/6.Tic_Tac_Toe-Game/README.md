# Tic Tac Toe Game (C++)

A console-based two-player Tic Tac Toe game implemented using object-oriented programming in C++.

## Features

* Two-player gameplay
* Dynamic game board
* Input validation
* Turn-based system

## Concepts Used

* Classes & Objects
* Vectors
* Game logic
* Conditional statements

## How to Run

Compile:

g++ TTT.cpp -o tictactoe

Run:

./tictactoe

## Learning Outcome

This project demonstrates how game logic can be implemented using object-oriented programming and data structures.
