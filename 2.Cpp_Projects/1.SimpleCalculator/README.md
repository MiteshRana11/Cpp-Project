# Simple Calculator (C++)

A console-based calculator application written in C++ that performs basic arithmetic operations using an object-oriented approach.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Menu driven interface

## Concepts Used

* Classes & Objects
* Functions
* Switch statements
* Encapsulation

## How to Run

Compile the program:

g++ SimpleCalculator.cpp -o calculator

Run the program:

./calculator

## Learning Outcome

This project demonstrates how object-oriented programming can be used to design modular and reusable code in C++.
