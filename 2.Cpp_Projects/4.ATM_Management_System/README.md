# ATM Management System (C++)

A console-based ATM simulation that allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money.

## Features

* User authentication with PIN
* Check account balance
* Deposit money
* Withdraw money
* Menu-driven interface

## Concepts Used

* Object-Oriented Programming
* Encapsulation
* Functions
* Conditional logic

## How to Run

Compile:

g++ AMS.cpp -o atm

Run:

./atm

## Learning Outcome

This project demonstrates how object-oriented design can be used to simulate a simple banking system.
