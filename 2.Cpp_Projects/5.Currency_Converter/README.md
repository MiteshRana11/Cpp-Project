# Currency Converter (C++)

A console-based application that converts currency values between different currencies using predefined exchange rates.

## Features

* Convert between multiple currencies
* Dynamic rate calculation
* Error handling for unsupported currencies

## Concepts Used

* STL containers (unordered_map)
* Exception handling
* Functions
* Data processing

## How to Run

Compile:

g++ CC.cpp -o converter

Run:

./converter

## Learning Outcome

This project demonstrates how to use STL containers and exception handling to build a flexible conversion system.
